# 燃动校园｜游戏化运动激励工具

这是一个面向大学生的静态网页原型，页面尽量按真实应用界面呈现：

- 今日：任务打卡、等级进度、战队能量、七日挑战
- 计划：课表排期、强度调节、校园路线、连续保护
- 战队：队友状态、小队挑战、助攻补位、战队动态
- 奖励：校园兑换、运动周报、徽章收集
- 部署稳定性：图标脚本已本地化，仓库包含 `.nojekyll` 以减少 GitHub Pages 处理差异

## 打开方式

直接打开 `index.html` 即可预览。

## 部署方式

可将整个 `gamified-fitness-tool` 文件夹上传到任意静态托管平台，例如 Netlify、Vercel、GitHub Pages、Cloudflare Pages 或 NoCode 的静态页面入口。部署成功后复制访问链接提交。

如果平台要求入口文件，选择：

```text
index.html
```

如果平台要求发布目录，选择：

```text
gamified-fitness-tool
```
