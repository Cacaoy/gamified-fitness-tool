# 燃动校园｜游戏化运动激励工具

这是一个面向测评题目的静态网页原型，重点展示产品结构与功能设计：

- 运动任务：日常、组队、考试周三类轻量任务
- 社交激励：寝室战队、班级榜单、队友补位
- 成就系统：等级经验、徽章墙、七日挑战地图
- 运营评估：冷启动节奏、核心指标、校园兑换池

## 打开方式

直接打开 `index.html` 即可预览。

## 部署方式

可将整个 `gamified-fitness-tool` 文件夹上传到任意静态托管平台，例如 Netlify、Vercel、GitHub Pages、Cloudflare Pages 或 NoCode 的静态页面入口。部署成功后复制访问链接提交。

如果平台要求入口文件，选择：

```text
index.html
```

如果平台要求发布目录，选择：

```text
gamified-fitness-tool
```
